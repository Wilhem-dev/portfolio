# Portfolio

Petit projet personnel présentant mes réalisations et mon parcours.

## Description

Site front-end en React pour afficher des sections : accueil, projets, expérience, contact.

## Installation

1. Installer les dépendances :

```
npm install
```

2. Lancer l'application en développement :

```
npm start
```

## Scripts utiles

- `npm start` : lance le serveur de développement.
- `npm run build` : compile pour la production.

## Structure

Les sources se trouvent dans le dossier `src/` avec des composants dans `src/components/`.
